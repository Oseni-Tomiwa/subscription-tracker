# Subscription Tracker API

A production-ready REST API for managing subscriptions.

## Features

- User Authentication (JWT)
- Secure Password Hashing
- CRUD Subscription Management
- MongoDB Database
- Automated Email Reminders
- Error Handling
- Deployment Ready

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Upstash
- Nodemailer
